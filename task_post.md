# 생일 축하 웹사이트 제작 프로젝트

이 프로젝트는 `c:\dev\hbd` 경로에 특별한 생일 축하 메시지를 담은 프리미엄 웹사이트를 구축하는 것을 목표로 합니다.

## 진행 단계

1.  **기획 및 구조 설정**: 웹사이트의 구성 요소와 디자인 테마를 결정합니다.
2.  **기본 파일 생성**: `index.html`, `style.css`, `script.js` 파일을 생성합니다.
3.  **디자인 구현**: 현대적이고 화려한 디자인(Glassmorphism, 애니메이션)을 적용합니다.
4.  **인터랙티브 요소 추가**: 폭죽 효과, 메시지 공개 등 재미있는 기능을 구현합니다.
5.  **내용 개인화**: 사용자로부터 받은 메시지로 내용을 채웁니다.

## 주요 기능
-   화려한 오프닝 애니메이션
-   카드 형태의 메시지 박스 (유리 질감)
-   폭죽(Confetti) 효과
-   반응형 레이아웃
