# 작업 현황: 모바일 최적화 및 조작 개선

## ✅ 완료된 작업
1.  **모바일 스와이프 조작 구현**
    - `script.js`에 위/아래 스와이프 감지 로직 추가
    - 위로 스와이프: 점프
    - 아래로 스와이프: 수그리기 (600ms 유지 후 자동 해제)
    - 짧은 터치: 점프 (기존 클릭 유지)
2.  **반응형 레이아웃 적용**
    - `style.css`에 미디어 쿼리(`max-width: 600px`) 추가
    - 게임 영역 높이 축소 (400px -> 300px)
    - 캐릭터 위치를 % 단위로 조정하여 화면 밖으로 나가지 않게 처리
    - 모바일 환경에 맞춰 폰트 및 버튼 크기 축소
3.  **UI 안내 문구 수정**
    - `index.html` 시작 화면에 스와이프 조작법 설명 추가
